# GamePass

---

An e-commerce storefront simulator for video games built with .NET Core 3.1

#### Deployment: [Azure](https://gamepassddl.azurewebsites.net/)

#### Third-party APIs:

-   [Bootstrap](https://getbootstrap.com/)
-   [Font Awesome](https://fontawesome.com/)
-   [DataTables](https://datatables.net/)
-   [Toastr Notifications](https://codeseven.github.io/toastr/)
-   [Sweet Alert](https://sweetalert2.github.io/)
-   [Tiny.cloud for textarea](https://www.tiny.cloud/docs/api/)
-   [Stripe Payment](https://stripe.com/docs/api)
